# 집사모여 — PRD

> **슬로건:** 우리 애 자랑하러
> **버전:** v1.0 (MVP)
> **작성일:** 2026-04-10

---

## 1. 서비스 개요

강아지·고양이를 키우는 집사들이 반려동물을 자랑하고 소통하는 커뮤니티 웹 서비스.

**목표:**
- 부담 없이 일상을 공유하고 자랑할 수 있는 공간
- 깔끔하고 직관적인 UI/UX
- 로그인 없이도 모든 기능 체험 가능 (낮은 진입 장벽)

---

## 2. 사용자 유형

**일반 회원**
- 소셜 로그인 (카카오, 네이버, 구글) OAuth 2.0
- 모든 기능 이용, 데이터 영구 저장

**둘러보기 유저**
- 진입 시 임시 계정 자동 생성 (UUID 기반 닉네임)
- 모든 기능 이용 가능 (글 작성, 좋아요, 댓글, 팔로우)
- 세션 종료 후 24시간 뒤 자동 삭제 (스케줄러)
- "회원가입하면 기록이 저장돼요!" 전환 유도 UI 노출

---

## 3. 핵심 기능

### 3.1 인증
- 소셜 로그인: 카카오, 네이버, 구글
- 둘러보기: 임시 계정 자동 생성
- 로그아웃, 회원 탈퇴 (관련 데이터 전체 삭제)
- 세션 기반 인증

### 3.2 유저 프로필

> 참고: [Grimity 프로필](https://www.grimity.com/aforedyogurt) 스타일

- 프로필 이미지 (기본 이미지 제공, S3 업로드)
- 닉네임 (필수, 중복 불가)
- 소개글 (선택)
- 작성 게시글 목록
- 팔로워/팔로잉 수 표시

### 3.3 게시글
- 구성: 제목 + 사진(필수, 최소 1장~최대 5장) + 내용
- 이미지 제한: JPG/PNG/WEBP, 10MB/장, 최대 5장
- 본인 게시글만 수정·삭제
- 상세: 이미지 슬라이더, 작성자 정보 → 프로필 이동

### 3.4 댓글
- 게시글 하단 텍스트 댓글
- 본인 댓글만 수정·삭제
- 최신순 정렬
- 대댓글은 추후 확장

### 3.5 좋아요
- 게시글 대상, 토글 방식
- 유저당 게시글 1회 (중복 방지)
- 좋아요 수 게시글 카드 및 상세에 노출

### 3.6 팔로우
- 팔로우/언팔로우 토글
- 팔로워/팔로잉 목록 조회
- 팔로우 피드는 추후 확장

### 3.7 메인 페이지

> 참고: [Grimity 메인](https://www.grimity.com) 레이아웃 스타일

**상단 네비게이션**
- 로고(집사모여) + 검색바 + 로그인/프로필

**메인 배너 영역**
- 오늘의 멍냥 (인기 게시글 하이라이트)
- 최근 24시간 좋아요 상위 N개, 1시간 단위 갱신

**최신 게시글 영역**
- 그리드 레이아웃 (사이드바 없음, 깔끔한 세로 흐름)
- 게시글 카드: 썸네일 + 제목 + 좋아요 수 + 작성자
- 무한 스크롤 또는 페이지네이션

### 3.8 검색
- 제목 기반 검색

## 4. MVP 우선순위

**P0 (필수)**
- 소셜 로그인 + 둘러보기
- 유저 프로필 (조회, 수정)
- 게시글 CRUD + 이미지 업로드
- 좋아요, 댓글 CRUD

**P1 (중요)**
- 팔로우/언팔로우
- 오늘의 멍냥
- 검색
